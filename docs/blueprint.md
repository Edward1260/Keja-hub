# **App Name**: NairobiPad

## Core Features:

- Student Property Discovery: Students can search, filter (by price, location, amenities), and view detailed listings for available student accommodations.
- Seamless Booking & M-Pesa Payments: Enables students to book a chosen property and complete transactions securely via integrated M-Pesa payment gateway.
- Landlord Property Management: Landlords can easily create, edit, and manage their rental property listings, including availability and property details.
- AI Price Optimization Tool: An AI tool provides landlords with data-driven recommendations for competitive property pricing based on market trends and property features.
- AI Student-Property Matching Tool: An AI tool recommends personalized property listings to students based on their profile, search history, and stated preferences.
- AI Fraud Monitoring Tool (Admin): An AI tool that assists administrators by detecting and flagging potentially fraudulent listings, user activities, or payment patterns for review.
- Secure User Accounts & PostgreSQL Integration: Foundational role-based user authentication and authorization for students, landlords, agents, and administrators, backed by a robust PostgreSQL database.

## Style Guidelines:

- Primary color: A vibrant, clear mid-tone blue (#2680D9), representing trust, intelligence, and a modern platform feel. It provides a solid foundation for interactive elements.
- Background color: A very light blue-gray (#F0F2F4), creating a clean and spacious canvas that visually complements the primary color while ensuring readability for extensive text.
- Accent color: A soft, yet distinctive lavender (#D98CD9) used for highlights, calls-to-action, and subtle expressive elements. It provides an analogous contrast without being overly bold.
- Headlines and prominent text: 'Space Grotesk' (sans-serif) to convey a modern, tech-savvy, and intellectual feel fitting for a smart platform. Body text and general content: 'Inter' (sans-serif) for its high legibility and clean, objective appearance across various screen sizes.
- Utilize a consistent set of geometric and minimalistic line icons to maintain a clean, modern aesthetic. Icons should clearly represent property types, search filters, and user actions.
- A responsive, grid-based layout for dashboards and listings, ensuring optimal readability and navigation across all devices. Emphasize generous white space to reduce clutter and highlight content.
- Incorporate subtle, tasteful animations for transitions between pages, hover states, and dynamic data loading. Animations should enhance user feedback and visual appeal without distracting from core functionality.
